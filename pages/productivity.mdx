# Productivity

- [Flowchart](https://flowchart.fun/) - web app that lets you easily create flow charts just via typing. Every single line of text is a node, indentation makes a child node. <nobr className="v-align"><i className="ri-html5-line"></i> <i className="ri-open-source-line"></i></nobr>
  ![Flowchart](/flowchart.webp)

- [Folder Peek](https://sindresorhus.com/folder-peek) - quickly access documents, files, folders, and apps from the menu bar. <nobr className="v-align"><i className="ri-apple-line"></i></nobr>

- [Hyperkey](https://hyperkey.app/) - convert the `caps lock` key or any modifier keys to the hyper key, all four modifiers combined: ⌃⌥⌘⇧. The hyper key acts as an additional modifier key that you can use in any app with keyboard shortcuts. <nobr className="v-align"><i className="ri-apple-line"></i></nobr>

- [Hungrymark](https://zhengying.github.io/hungrymark/) provides quick access to your collected URLs, local favorite files, folders, apps. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>

- [Itsycal](https://www.mowglii.com/itsycal/) - a tiny menu bar calendar: quickly have a month view and see your events with one click. <nobr className="v-align"><i className="ri-apple-line"></i></nobr>
  > Alternatives: [Dato](https://sindresorhus.com/dato) and [Day-O](https://shauninman.com/archive/2020/04/08/day_o_mac_menu_bar_clock_for_catalina).

- [Menuwhere](https://manytricks.com/menuwhere/) - pop up the frontmost app's menus at the current mouse location. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>
  ![Menuwhere](/menuwhere.webp)

- [Paletro](https://appmakes.io/paletro) - command palette to access any functionality of a currently opened application. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>

- [Soulver](https://soulver.app/) - a powerful but simple notepad calculator app for macOS that supports units, dates, currency and time zones conversions, variables and lines referencing what makes it easier than spreadsheets. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>
  ```bash 
  #Examples
  $300 in Euro  
  25% of 900  
  June 9 + weeks  
  6pm Sydney in Chicago  
  ```
  >  Alternatives:  
    [Numi](https://numi.app/) - a free calculator for Mac which works completely the same way: just type a string of what you need and get the result.  
    [Numbr](https://numbr.dev/): fewer features but works right in a browser.

- [Taskpaper](https://www.taskpaper.com/) - a plain text to-do list with outlining power. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>

- [Voiceliner](https://a9.io/voiceliner/) - a fast way to capture and store voice notes in a nesting structure. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-open-source-line"></i></nobr>

- [WorkFlowy](https://workflowy.com/b/) - a simple but very powerful way to stay organized with infinite nesting structure. My favourite tool.

- [Wox](http://www.wox.one/) is a free launcher for Windows. With an `Alt` + `Space` keyboard shortcut you can search for apps, files, folders and any queries on the internet. <nobr className="v-align"><i className="ri-windows-fill"></i> <i className="ri-open-source-line"></i></nobr>
