# Browsing

- [Anybox](https://anybox.cc/) - a modern native bookmark manager for mac and iOS. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>

- [Bumpr](https://www.getbumpr.com/) - select which app to open when clicking a web or mail link. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>
  > Alternatives:  
    [Browser Tamer](https://www.aloneguid.uk/projects/bt/) for Windows and [Velja](https://sindresorhus.com/velja) for macOS.

- [MenubarX](https://menubarx.app/) - a browser in your menu bar. <nobr className="v-align"><i className="ri-apple-line"></i></nobr>

- [Min](https://minbrowser.org/) - a fast, minimal and open source browser. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-windows-fill"></i> <i className="ri-ubuntu-line"></i> <i className="ri-open-source-line"></i></nobr>

- [qutebrowser](https://github.com/qutebrowser/qutebrowser) - a keyboard-driven, **vim**-like browser based on `Python` and `Qt`. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-windows-fill"></i> <i className="ri-ubuntu-line"></i> <i className="ri-open-source-line"></i></nobr>

- [TWP](https://addons.mozilla.org/pt-BR/firefox/addon/traduzir-paginas-web/) - translate current web page in real time using Google or Yandex. <nobr className="v-align"><i class="ri-firefox-fill"></i> <i className="ri-open-source-line"></i></nobr>
