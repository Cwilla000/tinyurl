# Text

- [Bob](https://bobtranslate.com/) - a macOS translation app for text and pictures, and manual input. The [PopClip](https://pilotmoon.com/popclip/) plugin is also available. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-open-source-line"></i></nobr>

- [CopyChar](https://copychar.cc/) - find and copy special characters to your clipboard. <nobr className="v-align"><i className="ri-html5-line"></i></nobr>

- [Copy Text Easily](https://devapt.com/copy-text-easily) - a simple way to copy paragraphs, sentences and custom selections on web pages in one click.

- [Espanso](https://espanso.org/) - a cross-platform text expander that detects a specific keyword and replaces it with something else. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-windows-fill"></i> <i className="ri-ubuntu-line"></i> <i className="ri-open-source-line"></i></nobr>

- [OwlOCR](https://owlocr.com/) - a handy text recognition app living in menubar. Just select an area and then paste text wherever you want. <nobr className="v-align"><i className="ri-apple-line"></i></nobr>
  > Alternative:  
  [TextSniper](https://textsniper.app/) extracts text from images and other digital documents.

- [PopClip](https://pilotmoon.com/popclip/) appears when you select text, giving access to a customizable row of +200 helpful actions. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>

- [Runestone](https://runestone.app/) - a lightweight and fast plain text editor with syntax highlighting, line numbers, theming and more. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-open-source-line"></i></nobr>
  ![Runestone on iPad](/runestone.webp)

- [Simple keyboard](https://github.com/rkkr/simple-keyboard) - an open source minimalist Android keyboard focused on just typing some text. No emojis, GIFs, spell checking and word prediction. <nobr className="v-align"><i className="ri-android-line"></i> <i className="ri-open-source-line"></i></nobr>

- [Table Tool](https://github.com/jakob/TableTool) - a simple CSV editor for macOS. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-open-source-line"></i></nobr>
  > Version (fork) for Apple Silicon can be found [here](https://github.com/rafi16jan/TableTool).
