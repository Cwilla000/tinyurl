# Files

- [age](https://github.com/FiloSottile/age) - a simple, modern and secure file encryption tool, format and Go library.

- [Broot](https://github.com/Canop/broot) - an alternative to the standard `tree` command, which allows you to view and navigate directories, perform fuzzy searches, run commands on the selected files or directories and etc.

- [Collision](https://apps.gnome.org/app/dev.geopjr.Collision/) compares and verifies MD5, SHA-256, SHA-512 and SHA-1 hashes. <nobr className="v-align"><i className="ri-ubuntu-line"></i></nobr>

- [Czkawka](https://github.com/qarmin/czkawka) - a simple and fast app which finds duplicates, empty folders, similar images etc. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-windows-fill"></i> <i className="ri-ubuntu-line"></i> <i className="ri-open-source-line"></i></nobr>

- [Default Folder X](https://www.stclairsoft.com/DefaultFolderX/) enhances the `Open` and `Save` dialogs in all of your applications. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>

- [Everything](https://www.voidtools.com) - a free Windows search engine that locates files and folders by name instantly. <nobr className="v-align"><i className="ri-windows-fill"></i></nobr>
    > Also check [EverythingToolbar](https://github.com/stnkl/EverythingToolbar) which adds **Everything** integration with the Windows taskbar.

- [Hat.sh](https://hat.sh/) is a web app that provides secure local file encryption in the browser. <nobr className="v-align"><i className="ri-html5-line"></i> <i className="ri-open-source-line"></i></nobr>

- [Hazel](https://www.noodlesoft.com/) - create rules to automatically organize files in many ways: open, move, rename, archive and tag files based on their name, date, type and much more. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>

- [Marta](https://marta.sh/) - native, dual-pane file manager for macOS. <nobr className="v-align"><i className="ri-apple-line"></i></nobr>
    ![Marta dual pane files manager](/marta.webp)

- [nnn](https://github.com/jarun/nnn) - a tiny, fast, full-featured terminal file manager.
  > Alternatives: [lf](https://github.com/gokcehan/lf) and [llama](https://github.com/antonmedv/llama) - CLI file managers written in Go.

- [QuickLook](https://pooi.moe/QuickLook/) allows to peek into a file content in lightning speed by just pressing the Space key. <nobr className="v-align"><i className="ri-windows-fill"></i> <i className="ri-open-source-line"></i></nobr>

- [uPic](https://github.com/gee1k/uPic) - a simple pictures and files upload tool with support of different hosting services. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i> <i className="ri-open-source-line"></i></nobr>

- [VanillaTreeViewer](https://abhchand.me/vanilla-tree-viewer/) - a minimalist file browser for compactly displaying several files at once. <nobr className="v-align"><i className="ri-html5-line"></i> <i className="ri-open-source-line"></i></nobr>

- [Yoink](https://eternalstorms.at/yoink/mac/) - improves drag & drop by providing a "shelf" for your files until you need them again. Yoink is also available as an [iOS app](https://apps.apple.com/app/yoink-improved-drag-and-drop/id1260915283?mt=8&at=1001l8pT) which stores anything you can drag, copy, share or download and syncs it across your devices. <nobr className="v-align"><i className="ri-apple-line"></i> <i className="ri-money-dollar-circle-line"></i></nobr>
  > Free alternatives: [Dropzone](https://apps.apple.com/in/app/dropzone-4/id1485052491?mt=12) for macOS and [Gladys](https://apps.apple.com/us/app/gladys/id1257526927) for iOS.

## Sharing

- [Snapdrop](https://snapdrop.net/) - a browser based AirDrop alternative. This handy app lets you share text and files between devices on the same network. <nobr className="v-align"><i className="ri-html5-line"></i> <i className="ri-open-source-line"></i></nobr>
  > You can also try [sharedrop.io](https://www.sharedrop.io/) which works the same way.

- [Transfer.sh](https://transfer.sh) - a free service for sharing files right from the command line.
  ```bash
  # upload
  $ curl -H "Max-Downloads: 1" -H "Max-Days: 5" --upload-file ./hello.txt https://transfer.sh/hello.txt

  # download 
  $ curl https://transfer.sh/66nb8/hello.txt -o hello.txt
  ```

- [Wormhole](https://wormhole.app/) - a simple and safe way to send files and folders. All your shared data is end-to-end encrypted and is automatically expired after 24 hours. Upload size is limited to 10GB. <nobr className="v-align"><i className="ri-html5-line"></i> <i className="ri-open-source-line"></i></nobr>
