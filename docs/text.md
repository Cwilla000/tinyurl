# Text

- [Espanso](https://espanso.org/) - a cross-platform open source text expander. It detects when you type a specific keyword and replaces it with something else.

- [OwlOCR](https://owlocr.com/) - a handy text recognition app living in menubar. Just select an area and then paste text wherever you want.
  :::note Alternative
  [TextSniper](https://textsniper.app/) extracts text from images and other digital documents.
  :::

- [Simple keyboard](https://play.google.com/store/apps/details?id=rkr.simplekeyboard.inputmethod&hl=en&gl=US) - an open source minimalist Android keyboard focused on just typing some text. No emojis, GIFs, spell checking and word prediction.
